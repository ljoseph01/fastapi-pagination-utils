# FastAPI pagination

Just a simple pagination utility for FastAPI calculations. Built specifically for SQLModel and FastAPI with Async database sessions. I haven't put a whole lot of effort into making this work generically, so it's unlikely to be useful in a lot of cases, but it does what I need to do. At the time of writing, it's small enough to not have to explain how to use it. Just a few lines of code to read and you should be able to figure it out.
